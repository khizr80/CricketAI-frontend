## Features

- Interactive dashboard for match predictions  
- Player and team statistics visualization  
- Match-by-match analytics  
- Responsive design compatible with desktop and mobile  
- Dynamic charts and graphs (e.g. bar charts, line charts)

## Tech Stack

- **Framework:** React (or Vue / Angular)  
- **UI Library:** Tailwind CSS / Bootstrap / Material UI  
- **State Management:** Redux / Context API  
- **Charts & Visualization:** Chart.js / D3.js / Recharts  
- **HTTP Client:** Axios / Fetch API
